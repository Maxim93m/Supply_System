# ПРИТОЧНАЯ УСТАНОВКА П93

В качестве базового контроллера автоматизации выбран программируемый логический контроллер Fastwel I/O CPM723 (программирование в среде CODESYS V3.5).

## Общие сведения
Объект автоматизации – приточная установка, включающая:

1.	Воздушный клапан с электроприводом 0–10В
2.	Фильтр G4	
3.	Фильтр G7
4.	Водяной нагреватель. Клапан 0–10В. Циркуляционный насос
5.	Электрокалорифер
6.	Секция охлаждения. Клапан 0–10В
7.	Пароувлажнитель
8.	Бактерицидная секция
9.	Вентилятор с частотным преобразователем 0–10В
